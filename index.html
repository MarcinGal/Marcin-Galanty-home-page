O MNIE

Witaj. Nazywam się Marcin Galanty i od kilku miesięcy zgłębiam tajniki front-end web developmentu. Swoją naukę rozpocząłem od kursów internetowych. Na nich poznałem podstawy htmla i cssa. Następnie pod koniec 2018 roku brałem udział w stacjonarnym intensywnym kursie Front-endu. Podczas 320 godzin programowania uczyłem się posługiwać: java scriptem, reactem, reduxem, firebasem, material-ui i gitem. W systemie scrum pracowałem też razem z 3 osobami nad aplikacją służącą do zarządzania i przeglądania wydarzeń. Można ją znaleźć tutaj W ramach kursu pracowaliśmy też za pomocą metody scrum nad naszym własnym projektem. Była to aplikacja "event planner" służąca do zarządzania wydarzeniami. Można ją znaleźć tutaj: home-page, application. Obecnie dalej doskonalę warsztat pracy w Java Scripcie i pracuję nad pierwszymi zleceniami aplikacji. Chętnie odpowiem na możliwość pracy jako Fron-end Developer w Lublinie oraz na zlecenia zdalne.

Technologie

HTML

HTML (Hypertext Markup Language) jest to kod używany do tworzenia struktury strony i jej zawartości. Na przykład treść może być uporządkowana w obrębie zestawu akapitów, listy punktowanych punktów lub tabel obrazów i danych.

CSS

Cascading Style Sheets, w skrócie CSS) – język służący do opisu formy prezentacji (wyświetlania) stron WWW. CSS został opracowany przez organizację W3C w 1996 r. jako potomek języka DSSSL przeznaczony do używania w połączeniu z SGML-em.

GIT

Git – rozproszony system kontroli wersji. Stworzył go Linus Torvalds jako narzędzie wspomagające rozwój jądra Linux. Git stanowi wolne oprogramowanie i został opublikowany na licencji GNU GPL w wersji 2.
